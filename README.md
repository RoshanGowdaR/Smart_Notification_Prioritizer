# 📌 Smart Notification Prioritizer

## 🚀 Overview
Smart Notification Prioritizer is an AI-based system that automatically classifies notifications into **Important** and **Non-Important** categories.  
The goal is to reduce digital distractions and help users focus on what truly matters.

---

## ❗ Problem Statement
Users receive a large number of notifications daily from emails, messaging apps, and other platforms.  
This leads to:
- Information overload  
- Reduced productivity  
- Missing important messages  

---

## 💡 Solution
Our system uses Machine Learning algorithms to analyze notifications and prioritize them based on their importance.

---

## 🧠 Features
- 📥 Collects notification data (Mock / Gmail API)
- 🔍 Extracts key information (sender, subject, keywords)
- 🤖 Classifies notifications using ML models
- 📊 Displays:
  - Important notifications ✅  
  - Other notifications 📂  
- ⚡ Simple and user-friendly UI

## 🔄 Working Process

1. Data Collection  
   - Mock notification dataset or Gmail API  

2. Preprocessing  
   - Extract features like:
     - Sender  
     - Message content  
     - Keywords  

3. Model Training  
   - Train ML models to classify notifications  

4. Prediction  
   - Classify notifications as:
     - Important  
     - Not Important  

5. UI Display  
   - Show prioritized notifications to the user
